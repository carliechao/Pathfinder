# Pathfinder - Memory Game
starter project and my first repo
this project is a game that tests your memory by playing leds in sequence and having you to repeat the pattern!

A little into the motivation of why I started this project was because I was interested in robotics but never designed anything myself. I had taken note of hackclub for a while but never decided to actually undertake in any of the programs until one day... My friends and I saw how grim our college apps were at the end of 11th grade and decided soemthing had to be done. To be brief, nothing was done. But I did not want to extinguish the light of the motivation just yet and continued with my project. Some of my disdain is echoed into the memory game where you continously fight the overwhelming torrent of forgetfulness by strengthening your mental fortitude through a trial of memory. 

Similar to the challenge that this memory game presents, the process of designing the board also took had lots of challenges I had to overcome and learn from. I learnt, for example, how to draw schematics, lay out my pcb, and code the thing. I am excited to finally finish up this project and see where this experience will take me next.
